# LlmChatWindowSetting設計書

## 概要
LlmChatWindowの設定パラメータを保持する構造体。UI関連とLLMの初期プロンプトを管理

## 目的・スコープ
- チャットウィンドウのUI設定を一箇所にまとめる
- LLMに渡すシステムプロンプトの保持

## ほかのクラスとの関係
- `LlmChatWindow` - この設定構造体を使用してチャットウィンドウを初期化

## このクラスが継承するクラス
無し(構造体)

## このクラスのコンストラクタ
デフォルトコンストラクタ

## このクラスのデストラクタ
デフォルトデストラクタ

## このクラスに含まれるデータメンバ
- `double input_area_height` - 入力エリアの高さ(ピクセル)。デフォルト値: 30.0
- `s3d::StringView system_prompt` - LLMに渡すシステムプロンプト(省略可)

無し
