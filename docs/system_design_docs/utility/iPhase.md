# iPhase設計書

## 概要

フェーズ (Phase) の共通インターフェースを示す抽象クラス。各フェーズ実装はこのインターフェースに従って `update()` と `draw()` を実装する。

## 目的・スコープ

共通のフェーズインターフェースを定義することで、`GameManager` やシーン管理実装がフェーズ間で一貫した呼び出し (update/draw) を行えるようにする。具体的なフェーズロジックは各実装クラスに委譲する。

## ほかのクラスとの関係

- `GameManager` - フェーズ操作（切り替え、update/draw 呼び出し）を行う
- 各具体的フェーズクラス（`IntroductionPhase`, `JobSearchPhase`, `WorkPhase` 等） - このインターフェースを実装

## このクラスが継承するクラス

なし（純粋仮想インターフェース）

## このクラスのコンストラクタ

デフォルト

## このクラスのデストラクタ

仮想デストラクタを持つ（`virtual ~iPhase() = default;`）

## このクラスに含まれるデータメンバ

なし

## このクラスに含まれる定数

なし

## このクラスに含まれるpublicメソッド

- `virtual void update() = 0` - 毎フレームの更新処理
- `virtual void draw() const = 0` - 毎フレームの描画処理

## このクラスに含まれるprivateメソッド

なし

## このクラスで参照するアセットの情報

なし

## このクラスが参照する仕様書の項目

- フェーズ遷移図、各フェーズの動作

## イメージ図

無し

## このクラスが使用されるフェーズ

インターフェースとして全フェーズで使用

## 特記事項・メモ

- フェーズ実装は `App::Scene` を継承する設計が別途ドキュメントに示されているが、`iPhase` は純粋仮想インターフェースとして存在する
