# DebugSystem設計書

## 概要
デバッグ表示を行う静的ユーティリティクラス。F10キーで表示/非表示を切り替え、FPS、マウス座標、現在のフェーズ名、日付、精神力を画面左下に表示する。また、デバッグ用のボタン（暗転解除、次フェーズへ、日付/精神力操作、フェーズ選択）を提供する。

## 目的・スコープ
開発中のデバッグ作業を支援するために、ゲームの内部状態をリアルタイムで確認し、操作できる機能を提供する。リリースビルドでは無効化される想定。

## ほかのクラスとの関係
- `BlackOutUI` - 暗転の解除操作を行う
- `PhaseManager` - 現在のフェーズ名取得、フェーズ遷移操作を行う
- `GameCommonData` - 日付と精神力の情報取得と操作を行う
- `FontManager` - デバッグ表示用のフォントを取得する

## このクラスが継承するクラス
なし（静的クラス）

## このクラスのコンストラクタ
インスタンス化を禁止（コンストラクタは削除）

## このクラスのデストラクタ
インスタンス化を禁止（デストラクタは削除）

## このクラスに含まれるデータメンバ
- `static inline bool s_enabled = true` - デバッグ表示が有効かどうかのフラグ
- `static inline int s_fps = 0` - 最終更新時のFPS
- `static inline Vec2 s_mousePos = Vec2(0, 0)` - マウス座標
- `static inline size_t s_selectedPhaseIndex = static_cast<size_t>(PhaseManager::GetCurrentPhaseType())` - デバッグ用フェーズ選択インデックス

## このクラスに含まれる定数
なし

## このクラスに含まれるpublicメソッド
- `static void Update()` - 毎フレームの更新処理。F10キーでトグル、FPSとマウス座標を更新する
- `static void Draw()` - 描画処理。情報表示とデバッグボタンを描画する

## このクラスに含まれるprivateメソッド
なし（すべてpublicメソッド内で完結）

## このクラスで参照するアセットの情報
なし

## このクラスが参照する仕様書の項目
- デバッグ機能

## イメージ図
無し

## このクラスが使用されるフェーズ
- 全フェーズ（デバッグビルド時のみ）

## 特記事項・メモ
- `#if defined(_DEBUG) || defined(DEBUG)` で囲まれ、リリースビルドでは無効化される
- F10キーで表示/非表示を切り替える
- 情報表示は画面左下、ボタン群は画面下中央、フェーズ選択UIは画面右下に配置される
