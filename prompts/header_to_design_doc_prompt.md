# ヘッダーファイルに合わせて設計書の内容を更新する

## 対応範囲

- Game/job_search_phase/ResumeUI

## 指示内容

- Game以下の対応範囲内のヘッダーファイルをすべて確認して、それが参照しているヘッダも確認して
- 各ヘッダーに対応するクラス設計書の内容を、ヘッダー内の処理に応じて修正して
- 各ヘッダーに対応するクラス設計書が無ければ、docs/system_design_doc_template.mdの規格に合わせて生成して
- 設計書の内容に変更があった場合、doc/game_spec.mdとdoc/architectural_design_doc.mdにも変更が必要かを検討して
- 変更の必要があれば、対応して報告をして
