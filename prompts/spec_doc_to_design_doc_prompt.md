# 仕様書と設計書テンプレートから、設計書を生成する

## タスクの概要

- コンテキストとして渡されたgame_spec.mdとそのファイルが間接参照、直接参照しているファイルを読み込み、設計書を生成してください。
- 作成する設計書のフォーマットは、コンテキストとして渡されたsystem_design_doc_template.mdのフォーマットを守ってください

## タスクの実行順序

1. game_spec.mdを読み込む
2. 仕様書が参照しているファイルを特定する
3. 参照しているファイルを読み込む
4. system_design_doc_template.mdを読み込む
5. 読み込んだ情報をもとに設計書を生成する

## 設計書について

- 設計書はMarkdown形式で作成してください。
- 設計書はC++とSiv3Dで実装するシステムやモジュールであることを想定して作成してください。
- 設計書は日本語で作成してください。
- 設計書には適宜クラス図やシーケンス図を含めてください。
- 設計書内のクラスやモジュールはシンプルな設計を心がけてください。
- 設計書は読みやすさを重視し、適宜箇条書きを使用してください。
- 設計書は適宜複数ファイルに分割してください。
